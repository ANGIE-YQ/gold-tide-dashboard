## 〇、一句话定位

**潮汐模型是将"潮汐法则交易框架"工程化为可量化、可验证、可交易的短期择时Alpha模型。**

核心思路：**不直接用手写规则做交易信号**（已证明无样本外优势），而是把框架的"结构描述能力"转化为**无前视特征层**，喂给XGBoost学习"特征→未来收益"的映射。

---

## 一、框架概览：从"潮汐思维"到"数学模型"

### 1.1 潮汐法则核心概念

| 原始概念 | 工程化实现 |
|---------|-----------|
| 潮汐命名双名法 | ZigZag(3×ATR)切大潮汐A/B/C + ZigZag(1×ATR)切小潮汐a/b/c |
| 12项动能评估 | 12个尺度不变比值特征(s1-s12)，每项0-1，综合0-100 |
| 渐变法则 | 同向大潮汐动能对比 → 渐弱/渐强 |
| 边界约束 | 通道带宽与潮汐高度比值 → 约束强度 |
| 动能脉冲式释放 | 能量=高度×时间，CV衡量释放均匀度 |
| 反向节操作 | 小潮汐拐点(高低点)处发出信号，conf_idx+1入场 |

### 1.2 为什么不用"规则"直接交易？

```
用户手写规则: 弱动能→反转 → 直接当BUY/SELL信号
   ↓ 验证结果
样本外方向准确率: 50.1%（≈硬币，无优势）
ATR风控净值: -52.1%（实盘亏钱）
   ↓ 根本原因
1. 潮汐确认有滞后(中位2根, p90=5根)
2. "弱→反转"是描述性规律，不是方向预测规律
3. 手写规则的阈值(mid=50, band=10)一滚动就退化(39.5%)
```

### 1.3 正确的范式

```
潮汐结构 + 12项动能 → 转成"无前视特征层" → XGBoost学习特征→未来收益
                                                      ↓
                                           Walk-Forward OOS AUC = 0.68
                                           年化夏普 = 2.54 (T3风险叠加)
```

## 二、模型架构(5层流水线)

```
┌─────────────────────────────────────────────────────────┐
│ Layer 1: 潮汐分段引擎 (gold_tide_engine.py)              │
│   ZigZag 3×ATR → 174个大潮汐 A-Z                        │
│   ZigZag 1×ATR → 707个小潮汐 a-z                        │
│   严格无前视: 只在回撤≥ATR×mult时确认拐点                 │
├─────────────────────────────────────────────────────────┤
│ Layer 2: 特征层 (gold_tide_features.py)                  │
│   31维原始特征 + 13维增强特征 = 44维(优化后40维)          │
│   每根bar可用，严格无前视                                 │
│   - 当前潮汐运行态: cur_dir/cur_energy_ratio/cur_oneway   │
│   - 上一潮汐完整态: prev_mom/prev_s1~s12                  │
│   - 最近拐点信息: lp_L/lp_H/lp_bars_since                 │
│   - 进行中段(新增): ip_days/ip_ret/ip_energy_trend        │
│   - 市场状态(新增): rsi_14/regime_bull/vol_regime         │
├─────────────────────────────────────────────────────────┤
│ Layer 3: ML预测 (gold_tide_ml.py + gold_tide_optimized.py)│
│   XGBoost (n=300, depth=3, lr=0.05)                      │
│   扩展窗口Walk-Forward (train_init=1500, test_len=400)   │
│   双模型集成: WF扩展 + 全量校准                           │
│   Platt Scaling 概率校准                                  │
├─────────────────────────────────────────────────────────┤
│ Layer 4: T3风险叠加 (gold_tide_t3.py)                     │
│   置信过滤 + 动能调制 + vol-targeting + 牛熊缩放 + ATR止损│
│   回撤从-21.5%砍到-14.0% (↓35%)                           │
├─────────────────────────────────────────────────────────┤
│ Layer 5: 信号输出 (run_pipeline.py / gold_tide_optimized) │
│   BUY/SELL/HOLD + 概率 + 置信度 + 动态仓位                │
│   + 市场状态(牛熊/波动/RSI/超买超卖)                      │
└─────────────────────────────────────────────────────────┘
```

## 五、模型边界与局限(诚实声明)

### 已验证的边界

1. **短期性**: 边缘在1-2周(fwd=5/10),fwd=20已跑输买持有
2. **单市场**: 沪金AU0为主,T4跨资产已初步验证但未全球覆盖
3. **无宏观因子**: 未纳入利率/美元/通胀, P3模块已就绪待网络环境
4. **日线量价无效**: 成交量特征已被验证为噪声(仅日线),分钟线量价可能有用
5. **成本敏感**: 0.20%/边仍盈利但已显著压缩,实盘滑点需额外安全垫

### 已知覆盖盲区

| 盲区 | 影响 | 解决方案 |
|------|------|---------|
| 大涨(ip_ret>15%): 0样本 | 大反弹时模型无历史经验 | 扩充含更多牛熊周期的数据 |
| 长进行段(ip_days>90): 0样本 | 超长下行后信号不可靠 | 人工干预/宏观因子辅助 |
| 高波动(vol>2×): 仅38样本 | 极端行情下置信不足 | vol-targeting已部分缓解 |

### 优化路线图

| 优先级 | 方向 | 预期提升 | 状态 |
|--------|------|---------|------|
| P0 | 概率校准(Platt Scaling) | Brier↓13% | ✅ 已完成 |
| P0 | 移除死特征(4个) | 模型更轻量 | ✅ 已完成 |
| P1 | 交叉特征(prev_dir×cur_dir等) | AUC +0.02~0.03 | 待做 |
| P1 | 在线再训练(每季度) | 防信号退化 | 待做 |
| P2 | 宏观因子融合(P3) | AUC +0.02~0.05 | 代码就绪 |
| P2 | 分钟线量价特征 | 未知(需验证) | 待数据 |
| P3 | 深度学习/Transformer | 未知(日线LSTM已证无效) | 待GPU+分钟线 |

## 六、运行指南

### 环境要求

```bash
pip install numpy pandas matplotlib scikit-learn xgboost
```

### 一键运行

```bash
# 完整流水线(验证+风险+信号+报告)
python run_pipeline.py

# 增强信号(日常使用,含市场状态/动态仓位/概率校准)
python gold_tide_optimized.py

# 重建特征层
python gold_tide_optimized.py --rebuild

# 仅信号简报
python run_pipeline.py --quick
```

### 输出解读

| 输出 | 含义 | 如何用 |
|------|------|--------|
| P(涨) | 未来10日上涨的概率 | >0.55做多, <0.45做空 |
| 置信度 | |P-0.5| | >0.15强信号, >0.08中信号 |
| 市场状态 | 牛熊/波动/超买超卖 | 调整仓位大小 |
| 动态仓位 | 波动率调整后的建议仓位 | 低波动加仓,高波动降仓 |
| WF参考 | Walk-Forward验证概率 | 与实际P对比判断可靠性 |
