## 一、这个模型是干什么的？

**一句话：用AI帮你判断未来1-2周黄金会涨还是跌。**

它不是预言机——不会说"明天一定涨"。它给的是一个**概率**。比如"P(涨)=0.69"，意思是：根据18年历史中类似的情况，有69%的概率未来10天金价会涨。

你打开看板，看到的东西就三样：

| 你看什么 | 什么意思 | 怎么用 |
|---------|---------|--------|
| **P(涨) 数值** | 模型判断未来10天上涨的概率 | 大于0.55考虑做多，小于0.45考虑做空 |
| **置信度** | 模型对自己判断的确信程度 | 高于0.15是比较确定的信号 |
| **止损/目标价** | 基于当前波动率算出的安全边界 | 亏到这个价位就退出，涨到这个价位考虑止盈 |

## 二、核心思路：把金价想象成海浪

### 海浪有三层

站在海边看浪，你能看到：

- **大浪**（涨几个月的大行情）—— 模型叫"大潮汐"
- **小浪**（大浪里几天的波动）—— 模型叫"小潮汐"  
- **回头浪**（跌多了反弹一下，涨多了回调一下）

海浪不是杂乱无章的——它的起伏有规律：
- 一个浪头比上一个浪头低 → 潮在退
- 浪打来时干脆利落 → 动能强
- 浪打来时磨磨唧唧、反复拍 → 动能弱

### 模型做了什么

```
第一步：把18年的金价切成一段段"浪"
       ┌──大浪A──┐┌─大浪B─┐┌──大浪C──┐
       │a→b→c→d││a→b→c││a→b→c→d→e│  ← 每段小浪有个编号
       
第二步：给每段大浪打分（12个维度）
       · 涨得干脆吗？（s3一字型）
       · 浪头比上一浪大还是小？（渐变法则）
       · 波动幅度均匀吗？（s8均匀释放）
       · 从头到尾花了多久？（s4节数长度）
       ... 共12项，每项0-1分，综合0-100分

第三步：AI学习"什么样的浪接下来会涨"
       把4500天的数据喂给AI
       AI自己找出：
       "当上一段是下跌浪、当前浪动能衰减、RSI超卖时
        → 接下来10天有69%概率上涨"
```

### 为什么不让AI直接看K线？

试过。准确率只有50%——跟抛硬币一样。

因为直接看K线，AI看到的只是一堆数字的涨跌。而"潮汐"这个概念，把数字变成了**有结构的浪**：这浪多大、多快、是第几个浪、跟上一个浪比是强了还是弱了——这些才是真正有预测力的信息。

> **关键转折**：把"弱动能→买"这种简单规则扔掉，改为"把潮汐特征喂给AI，让AI自己找规律"。这个改动让准确率从50%提升到60%。

## 三、效果怎么样？（诚实数据）

### 预测准确率

用严格的"盲测"方法（训练数据和测试数据完全独立，不偷看答案）：

| 预测多久 | 准确率 | 可靠吗 |
|---------|--------|--------|
| 未来5天 | 61% | 比较可靠 |
| 未来10天 | **60%** | 主要用这个 |
| 未来20天 | 53% | 接近随机，不可靠 |

> 对比：因为金价长期上涨略多于下跌，闭眼猜"会涨"的正确率约55%。模型比闭眼猜高出约5个百分点。

### 模拟交易成绩

在实际可执行的条件下（含手续费、止损止盈）：

| 场景 | 年化收益 | 最大亏损幅度 |
|------|---------|------------|
| 模型交易（低手续费） | +49% | -14% |
| 模型交易（高手续费） | +35% | -16% |
| 买入不动 | +10% | — |

### 牛市熊市都能用吗？

能。把数据按牛市和熊市分开测：

| 市况 | 准确率 |
|------|--------|
| 牛市期间 | 83% |
| 熊市期间 | 83% |

两个数字几乎一样——说明它不是靠"赌金价永远涨"赚的钱。

## 四、看板上每个模块说明

打开看板，从上到下：

### 顶部：实时价格 + 模型数据
- 右上角：**实时金价**（每秒更新，来自交易所）
- 左边三格：模型用的收盘价、20日均线、当前波动率

### 核心信号区（最重要的部分）
- **大字 P(涨)**：0到1之间的数字。>0.55看多，<0.45看空
- **置信度**：模型有多确定。>0.15红色，0.08-0.15黄色，<0.08灰色
- **止损/目标**：模型建议的安全边界
- **仓位**：根据当前波动率自动计算的建议仓位

### 市场状态
- **牛/熊**：当前金价高于还是低于250日均线
- **波动率**：当前波动是正常水平的几倍。低于0.7可以适当加仓
- **RSI**：超买超卖指标。低于30说明跌过头了
- **vs 50日均线**：当前价格离均线多远。偏离越大，回归引力越强

### 图表区域
- **潮汐动能图**：蓝色=上涨浪，红色=下跌浪。柱子越高动能越强。注意相邻同颜色的柱子：如果越来越矮，说明动能衰减
- **进行中段图**：当前正在进行但还没结束的这一段。标注了每小段的高度和"能量"(高度×时间)。如果相邻同方向的小段能量越来越小，趋势可能在衰竭
- **价格走势图**：近60天价格曲线。蓝色段=上涨，红色段=下跌，标了每段的编号

## 五、常见问题

### Q: 看多但金价在跌，该信吗？
看"置信度"。置信度低（<0.08）时模型也在犹豫，建议观望。置信度高时历史上准确率也更高。

### Q: 目标价和止损价怎么来的？
基于ATR（平均真实波幅）——当前市场每天大概波动多少钱。止损设2.5倍ATR，目标设3倍ATR。波动大时这两个数字会自动变大。

### Q: 为什么信号有时一天变好几次？
模型每天只更新一次（收盘后）。但实时金价会变——如果金价盘中剧烈波动穿过了止损/目标位，以你自己的风控为准。

### Q: 能用来做日内交易吗？
不能。模型预测的是未来10个交易日，不是当天。

### Q: 为什么不看新闻和利率？
技术上可以做，但还没接入。目前模型只看价格形态。黄金受美元利率影响很大——以后版本会加入。

## 六、它的边界（诚实地说）

1. **只预测短期**。别拿它判断"下个月金价会不会破新高"——它看不到那么远。
2. **只看价格**。不知道美联储说了什么、中东发生了什么。
3. **过去不等于未来**。2008-2026年的规律可能在未来失效。
4. **10次里有4次是错的**。60%准确率意味着每10次预测有4次方向错误。**一定要设止损。**

## 七、进阶：想了解技术细节？

### 数据流

```
沪金AU0日线 ─→ 潮汐分段 ─→ 12项动能评分 ─→ 特征矩阵 ─→ AI训练 ─→ 概率预测
   (新浪API)    (ZigZag)     (s1-s12)      (40维)    (XGBoost)   (P值)
```

### 为什么"手写规则"不行？

最初版本用的是"动能弱了就买"这种简单规则。验证结果：
- 整体准确率：50.1%（= 抛硬币）
- 模拟交易：亏52%
- 换个参数就失效

**根本原因**：潮汐确认有延迟（平均滞后2天），在拐点处"当下"根本不知道这是拐点。手写规则等于在"事后诸葛亮"。

### 为什么AI可以？

AI不是记住"这种情况该买"，而是学习了几千种"浪的特征组合"对应的结果分布。当新情况出现时，AI说："历史上出现过类似的浪型100次，其中69次接下来涨了"——这比简单规则可靠得多。

### 技术栈
- Python + XGBoost + scikit-learn
- 特征层：严格无前视（训练时绝不偷看未来数据）
- 验证：扩展窗口 Walk-Forward（业界标准方法）
- 校准：Platt Scaling（让概率更准确）
- 部署：GitHub Pages + GitHub Actions
